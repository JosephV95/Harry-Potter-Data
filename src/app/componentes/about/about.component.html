Información sobre la página


Web desarrollada en Angular.

Se utilizó una API de Harry Potter disponible aquí. En ella se encuentra la información de los personajes como también una lista de hechizos con su descripción correspondiente.

De manera local se editó y agrego información en algunos personajes, para disponer de mayor cantidad de imágenes.
