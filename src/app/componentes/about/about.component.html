Información sobre la página

Web desarrollada utilizando la API de Harry Potter disponible en: https://hp-api.onrender.com.

La información de los personajes se consume mediante una petición GET desde: IR A API https://hp-api.onrender.com/api/characters, pero se edito de forma local para que se disponga con mayor cantidad de imagenés de personajes.

Y la lista de hechizos se extrajo desde: https://hp-api.onrender.com/api/spells.

Desarrollada con Angular
